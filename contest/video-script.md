# Video Script: sofIA Contest Entry

**Format:** Presenter (P) speaking to camera, interspersed with B-roll/Screen Recordings (SR) of the CLI in action.
**Duration:** ~2 Minutes

---

### **0:00 - 0:20 | Intro: The Gap**

**Visual:**

- **Camera:** Presenter standing in a modern office or clean home studio.
- **Graphic overlay:** "Idea" icon on one side, "Code" icon on the other, with a broken bridge in between.

**Audio (Presenter):**
"We all know the struggle. You finish an AI Discovery workshop, the energy is high, the whiteboards are full... and then? Silence. It takes weeks to turn those sticky notes into specs, and even longer to get a single line of code written. We call this 'The Gap'—where great AI ideas go to die."

### **0:20 - 0:40 | Meet sofIA & Internal Context**

**Visual:**

- **SR (Screen Recording):** A sleek terminal window opens. User types `npx sofia`. The logo animates.
- **SR:** The CLI shows `[WorkIQ] Connected`. Steps scroll by showing "Analyzing Outlook...", "Reading Meeting Notes...".
- **Camera:** Cut back to Presenter for a second, then back to screen.

**Audio (Presenter):**
"Enter sofIA. It’s an agentic CLI built with the GitHub Copilot SDK that facilitates the entire journey. And it starts with context.
Using **WorkIQ**, sofIA doesn't just ask you questions—it securely reads your emails, meeting notes, and internal docs. It understands your business reality involves 'Supply Chain delays' before you even type a word."

### **0:40 - 1:00 | External Intelligence with Foundry**

**Visual:**

- **SR:** Terminal shows `[Foundry] Searching Web...`.
- **SR:** A list of bullet points appears in the terminal citing external market trends and competitor analysis.
- **Overlay:** "Powered by Microsoft Foundry".

**Audio (Presenter):**
"But internal data isn't enough. sofIA reaches out to the world using **Microsoft Foundry** for web search. It validates your ideas against real-time market trends and technological capabilities, ensuring your use cases aren't just relevant to _you_, but viable in the market."

### **1:00 - 1:25 | Ideation & Selection**

**Visual:**

- **SR:** Fast-forward animation of the "AI Envisioning Cards" phase. Text scrolling rapidly.
- **SR:** A table appears: "Score Matrix". One idea is highlighted in Green: _Predictive Inventory Management_.
- **SR:** Text appears: "Selected Idea: Predictive Inventory Management. Reason: Highest ROI/Complexity ratio."

**Audio (Presenter):**
"It guides you through the full AI Discovery methodology—brainstorming, mapping workflows, and applying AI Envisioning Cards.
And here’s the kicker: It doesn't leave you paralyzed with choices. sofIA autonomously analyzes feasibility versus value and _selects_ the best candidate to build right now."

### **1:25 - 1:45 | The Ralph Loop (Generating Capability)**

**Visual:**

- **SR:** The terminal speeds up. Text colors change.
- **SR:** We see `> Generating Plan...`, `> Scaffolding Next.js App...`, `> Running Tests... (Failed)`, `> Fixing Code...`, `> Running Tests... (Passed)`.
- **SR:** A file tree expands in the sidebar showing `src/`, `components/`, `tests/`.

**Audio (Presenter):**
"Then, it builds it. Using an iterative 'Ralph Loop,' sofIA generates a complete implementation plan and starts writing code. It writes the app, writes the tests, runs them, fails, fixes its own bugs, and iterates until the code is green. Zero human touch."

### **1:45 - 2:00 | The Deliverable & Outro**

**Visual:**

- **SR:** Split screen: On the left, a Markdown file `specs/analysis.md` is open. On the right, a local host web browser shows the generated dashboard running.
- **Camera:** Presenter smiles.

**Audio (Presenter):**
"In under 20 minutes, you walk away with a comprehensive analysis document _and_ a working, test-covered Proof of Concept repository.
sofIA: Stop guessing. Start delivering."
