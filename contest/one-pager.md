## **Left Column: The Challenge**

**"The Workshop Gap"**

- Sticky notes don't compile.
- Weeks lost translating business intent to technical specs.
- "Blank page paralysis" when starting PoCs.

## **Center Column: The Solution (sofIA)**

**An Agentic Facilitator powered by GitHub Copilot SDK**

1.  **Context & Research**
    - **Internal:** **WorkIQ** analyzes Microsoft 365 data (Emails, Teams, Docs).
    - **External:** **Microsoft Foundry** powers web search for market/tech validation.

2.  **Methodology**
    - **12-Step Process:** Guided Discovery → Ideation → Planning.
    - **AI Envisioning Cards:** Automated card selection & scoring.
    - **Auto-Selection:** Agent rationally selects the best idea to build.

3.  **Execution (The "Ralph Loop")**
    - **Plan:** Generates architecture & specs.
    - **Code:** Iterative Write → Test → Fix cycle.
    - **Deliver:** outputs a full **PoC Repository** + **Analysis Docs**.

## **Right Column: The Impact**

- **300% Faster** Time-to-Prototype.
- **Zero** Context Loss between Business & Dev.
- **RAI Native:** Guardrails built into every step.
